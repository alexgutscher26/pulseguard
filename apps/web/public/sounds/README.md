# Audio Assets

This directory contains sound effects for the application.

## Required Files

- `click.mp3` - Standard button click
- `hover.mp3` - Subtle hover effect
- `success.mp3` - Positive action confirmation
- `error.mp3` - Error/Alarm sound
- `glitch.mp3` - Static/Glitch noise
